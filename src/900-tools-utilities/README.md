Welcome to the Tools and Utilities components section. This grouping contains development and operational tools that support Edge AI platform development, testing, and operations.

## Overview

The 900-tools-utilities components (900-999 range) provide essential development and operational tools including MQTT testing utilities, video processing tools, monitoring utilities, and development aids. These components support both edge and cloud development workflows.

## Components

| Component                                          | Description                                                                           |
|----------------------------------------------------|---------------------------------------------------------------------------------------|
| **[900-mqtt-tools](./900-mqtt-tools/README.md)**   | MQTT client tools and utilities for testing and monitoring MQTT communications        |
| **[901-video-tools](./901-video-tools/README.md)** | Video processing utilities including video-to-GIF conversion for captured media files |

## Framework Support

Each component supports multiple frameworks and languages:

- **Python** - Python scripts and utilities
- **Node.js** - JavaScript/TypeScript tools and CLI utilities
- **Rust** - High-performance CLI utilities and video processing tools
- **Shell** - Bash/PowerShell scripts for automation

## Getting Started

1. **Review Requirements**: Check individual component README files for prerequisites
2. **Install Dependencies**: Install required tools and dependencies per component
3. **Configure Environment**: Set up environment variables and configuration files
4. **Run Tools**: Execute tools according to component documentation

## Prerequisites

- Python 3.8+ for Python-based tools
- Node.js 18+ for JavaScript/TypeScript tools
- Rust toolchain for Rust-based CLI utilities
- Bash or PowerShell for script execution

For more information about the overall source code structure, see the [main source documentation](../README.md).
