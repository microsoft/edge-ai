# Tools and Utilities

## Overview

The Tools and Utilities grouping (900-999 range) contains standalone CLI tools, scripts, and supporting utilities for edge AI deployments. These components provide essential development and operational tools for working with edge infrastructure and applications.

## Components

### 900-mqtt-tools

MQTT testing and diagnostic tools for IoT Operations.

Location: [src/900-tools-utilities/900-mqtt-tools/](./900-mqtt-tools/)

### 901-video-tools

Video processing utilities including video-to-GIF conversion for captured media files.

Location: [src/900-tools-utilities/901-video-tools/](./901-video-tools/)

## Usage

Each component provides independent tooling with its own documentation and dependencies. Refer to individual component README files for installation instructions and usage examples.
