# AKS Integration Module

Deploys Azure Machine Learning extension on AKS cluster and
attaches it to Azure ML workspace as compute target for
ML training and inference workloads.

## Requirements

| Name      | Version         |
|-----------|-----------------|
| terraform | >= 1.9.8, < 2.0 |
| azapi     | >= 2.3.0        |
| azurerm   | >= 4.51.0       |

## Providers

| Name      | Version   |
|-----------|-----------|
| azapi     | >= 2.3.0  |
| azurerm   | >= 4.51.0 |
| terraform | n/a       |
