# Azure Kubernetes Service

Deploys a Kubernetes cluster in Azure with a system-assigned managed identity.
Supports private clusters with optional private endpoints and DNS zone management.

## Requirements

| Name | Version |
|------|---------|
| terraform | >= 1.9.8, < 2.0 |

## Providers

| Name | Version |
|------|---------|
| azurerm | n/a |
