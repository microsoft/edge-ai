# Messaging

Creates Azure IoT Operations dataflows for messaging scenarios including
Event Hub and Event Grid endpoints for edge-to-cloud data transmission.

## Requirements

| Name      | Version         |
|-----------|-----------------|
| terraform | >= 1.9.8, < 2.0 |
| azapi     | >= 2.3.0        |
| azurerm   | >= 4.51.0       |
