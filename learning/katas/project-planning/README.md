## Troubleshooting Guide

**Common Issues:**

- **Chatmode not responding properly**: Verify correct chatmode file path and restart VS Code if needed
- **Incomplete scenario generation**: Provide more specific business context and technical constraints in prompts
- **Weak capability mappings**: Focus on connecting specific business outcomes to technical Edge AI features
- **Planning session gets stuck**: Use more targeted prompts and break complex planning into smaller, manageable steps
- **Documentation quality issues**: Reference existing project examples and templates for structure and detail level

**Tips for Success:**

- Start with clear business context and specific technical constraints
- Use iterative approach - refine scenarios through multiple planning sessions
- Leverage existing scenario templates and examples for guidance
- Focus on concrete business outcomes rather than abstract technical features
