## 🎓 Accelerate Your Learning

**New to edge AI development?** Our [Learning Platform](../../learning/) provides hands-on training:

- **🥋 [Katas](../../learning/katas/)**: Focused practice exercises (**15-45 minutes**)
- **🧪 [Training Labs](../../learning/training-labs/)**: Comprehensive hands-on experiences (**2-8 hours**)
- **🤖 AI Coaching**: Built-in VS Code coaching prompts for discovery-based learning

**🚀 Quick Start Documentation & Learning:**

```bash
npm run docs
```

This opens the documentation with both the main docs and the interactive **Learning** tab, where you'll find:

- **🥋 Katas**: Focused practice exercises
- **🧪 Training Labs**: Comprehensive hands-on experiences
- **🤖 AI Coaching**: Built-in VS Code coaching prompts

> 💡 **Tip**: Once the docs open, click on the **"Learning Platform"** section in the sidebar to access all learning materials.

## 📋 Quick Reference

### Essential Prerequisites

- **Azure subscription** with appropriate permissions
- **GitHub account** with access to this repository
- **Docker Desktop** installed and running
- **Visual Studio Code** with Dev Containers extension
- **[HVE Core extension](https://marketplace.visualstudio.com/items?itemName=ise-hve-essentials.hve-core)** — provides AI prompts, agents, and coding instructions (auto-installed in Dev Container)
